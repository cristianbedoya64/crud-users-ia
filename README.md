# UARP-AI

Plataforma web para gestión de usuarios, roles y permisos, con módulos de IA y Data Science.

## Estructura
- `frontend/`: React + Vite + TailwindCSS
- `backend/`: Node.js + Express
- `ia/`: Python para IA y Data Science
- `docs/`: Documentación y diagramas

# IA y Data Science

Este módulo contiene scripts y utilidades para el procesamiento de datos, entrenamiento de modelos de IA y análisis avanzado.

## Estructura
- `db_config.py`: Configuración de conexión a la base de datos.
- `ia_panel.py`: Lógica principal del panel de IA.
- `inspect_db.py`: Inspección y análisis de la base de datos.
- `seed_ia_panel.py`: Carga de datos iniciales para IA.
- `test_db_connection.py`: Pruebas de conexión a la base de datos.
- `train_ia_model.py`: Entrenamiento de modelos de IA.

## Uso
Consulta cada script para instrucciones específicas. Revisa la documentación en `docs/` para detalles de arquitectura y flujos de datos.
